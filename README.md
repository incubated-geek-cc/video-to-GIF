  # 🎞️ video-to-GIF

  ### 🛠️ Converts video clips (.mp4) to gif (.gif) files. Uses HTML5 Canvas to render image graphics.

**🔌 An offline in-browser setup.**

<div align="left">

### 💻 Main JavaScript Plugins

</div>
<div align="left">
<ul>
	<li>GIFEncoder.js</li>
	<li>LZWEncoder.js</li>
	<li>NeuQuant.js</li>
</ul>
